2. List of all prime numbers from 1 to 100.
print([i for i in range(1,101) if len([j for j in range(1,i+1) if i%j==0])==2])

3. Read an input string & check if it's Palindrome.
print((s:=input())==s[::-1] and len(s)>=1)
